Análisis de tendencias

El análisis de tendencias se utiliza como primera aproximación a la previsión de la demanda de mano de obra, tanto para toda la plantilla como para otras categorías profesionales.

Se basa en las necesidades anteriores de empleados para predecir las necesidades futuras, para ello se toma una serie histórica representativa del nivel de empleo de por ejemplo 3 ó 5 últimos años y observar las tendencias en el nivel de empleo, para así estimar las necesidades para los próximos años en el caso de aceptar estas tendencias como válidas.

Estas tendencias del nivel de empleo pueden representarse gráficamente al alza, a la baja u otras

‹ ›

Cuando se estiman las necesidades de personal a corto plazo se pueden tomar las últimas 10 o 12 observaciones de la serie histórica ya que tienen mayor capacidad de predicción que las primeras.

El análisis de tendencias permite ver la tendencia, pero por sí solo no permite una estimación de las necesidades, esta técnica es el punto de partida para poder aplicar la técnica de medias móviles, que es una de las técnicas que se pueden aplicar para realizar la extrapolación que nos permite obtener la estimación que buscamos.
