Técnicas auxiliares del proceso de dirección estratégica

Objetivos

Mostrar las técnicas auxiliares más relevantes que se utilizan en el proceso de dirección estratégica de RRHH, con el objetivo de analizar el personal de la empresa y su importancia para obtener ventajas competitivas, objetivos y estrategias.

Adquirir las habilidades de análisis, manejo de información, resolución de problemas, y toma de decisiones a partir de las técnicas explicadas a continuación.

Procedimiento

La práctica está diseñada para hacerla íntegramente en el ordenador de forma que tanto la explicación como la evaluación se realizará en la misma plataforma.

El ejercicio consiste en una explicación de las 3 técnicas junto con dos ejercicios de prueba tipo test (no constan para la nota), tras estos podrá hacer la evaluación definitiva para la práctica.

La fecha tope para la realización de la práctica es el 5 de diciembre.
