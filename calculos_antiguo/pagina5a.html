Análisis de correlación-regresión

Se centra en el grado de correlación entre 2 variables:

X:

La Serie histórica del nivel de empleo.

Y:

Algún indicador de actividad tales como nivel de producción, volumen de ventas, volumen de trabajo, etc. Es una variable causal del nivel de empleo, esta variable causal depende del tipo de actividad de la empresa

Una vez elegidas las variables hay que obtener el coeficiente de correlación de Pearson entre ellas. Se hace dividiendo la covarianza de ambas variables entre el producto de las desviaciones típicas de cada variable.

Es importante tener en cuenta que la validez de esta técnica depende del coeficiente de correlación entre la variable causal (el indicador de actividad) y la variable a predecir (nivel de empleo). Si esta correlación no es válida, no se puede aplicar este análisis con esas variables.
